# Energy Overload Prediction using Machine Learning

## Overview
This project predicts energy overload scenarios using real-world smart grid data from ISO New England (ISONE).  
The goal is to identify potential overload conditions to support smarter and more stable energy management.

---

## Dataset
ISONE Smart City Energy Dataset (Kaggle):
https://www.kaggle.com/datasets/datasetengineer/isone-smart-city-energy-dataset

The dataset includes:
- Energy consumption and grid metrics
- Renewable generation (solar, wind)
- Weather conditions
- Urban activity data

Dataset is not included due to size constraints.

---

## Methodology
- Data cleaning & preprocessing  
- Exploratory Data Analysis (EDA)  
- Feature engineering  
- Machine Learning model training  

---

## Tools
Python | Pandas | NumPy | Scikit-learn | Matplotlib

## Results
The model captures patterns related to energy overload scenarios and provides predictive insights based on historical data.
